# 📊 Data Professional Survey Dashboard | Power BI

Developed an interactive Power BI dashboard to analyze survey responses from data professionals across different countries and job roles. The dashboard provides insights into salary trends, work-life balance, job satisfaction, programming language preferences, and career challenges using interactive visualizations and DAX measures.

# 🔹 Key Features:
• Data cleaning and transformation using Power Query
• Interactive dashboard with slicers and filters
• KPI Cards, Bar Charts, Treemaps, Donut Charts, and Gauge Charts
• DAX calculations for business insights
• User-friendly and responsive dashboard design

# 🛠️ Tools & Technologies:
• Power BI Desktop
• Power Query
• DAX
• Data Modeling

# 📈 Insights Generated:
• Average salary by job title
• Work-life balance and salary satisfaction
• Favorite programming languages
• Average age of survey participants
• Difficulty in entering the data industry
• Country-wise survey analysis


# 📌 This project demonstrates skills in data visualization, business intelligence, data transformation, and dashboard development using Microsoft Power BI.
